## Panoramica

Il **YouTube AI Dream Team** e un sistema di automazione avanzato che utilizza 39 agenti AI specializzati per gestire ogni aspetto di un canale YouTube. Ogni agente ha un SOP (Standard Operating Procedure) dettagliato che definisce il suo ruolo, input/output e logica operativa.

| Capacita | Descrizione |
|---|---|
| **Strategia & Ideazione** | Analisi nicchie, competitor, keyword SEO, titoli virali |
| **Pre-Produzione** | Script, storyboard, hook, CTA, shot list |
| **Produzione** | Editing, audio, thumbnail, sottotitoli |
| **Distribuzione** | Upload, Shorts, cross-platform, newsletter |
| **Engagement** | Commenti, sentiment, superfan, collaborazioni |
| **Monetizzazione** | Lead gen, sponsorship, A/B testing, analytics |
| **Orchestrazione** | Analisi video, workflow n8n/Make, memoria GitHub |
| **Next-Gen AI** | Multi-agent, deep research, voice cloning, creative AI |

## Quick Start

### 1. Fork & Clone

```bash
git clone https://github.com/YOUR_USERNAME/youtube-ai-dream-team-open.git
cd youtube-ai-dream-team-open
```

### 2. Setup del tuo GitHub Token

```bash
# Crea un Personal Access Token su GitHub:
# Settings > Developer settings > Personal access tokens > Fine-grained tokens
# Permessi necessari: Contents (Read & Write), Metadata (Read)

export GITHUB_TOKEN="your_github_pat_here"
```

### 3. Carica su un tuo repo privato

```bash
# Crea un repo privato su GitHub
git remote add private https://github.com/YOUR_USERNAME/youtube-ai-dream-team.git
git push private main
```

### 4. Usa il sistema

Apri una chat con un AI (ChatGPT, Claude, Gemini, ecc.), incolla il contenuto di `dream_team_index.md` e inizia a usare gli agenti.
